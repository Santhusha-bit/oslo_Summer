# Oslo → Europe 🗺️
### Cheapest Schengen Weekend Travel Plan

A React web app showcasing a budget travel plan for students based in Oslo who want to explore Europe on weekends — **8 countries, 6 weekends, ~$875–$950 total**.

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

```bash
git clone https://github.com/YOUR_USERNAME/oslo-schengen-travel-planner.git
cd oslo-schengen-travel-planner
npm install
npm run dev
```

App will run at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Deploy to GitHub Pages
- Push to `main`. The GitHub Actions workflow will build and publish the Vite `dist/` output to GitHub Pages.
- Ensure your repo has GitHub Pages set to **Source: GitHub Actions**.

## 💡 About This Plan

Built for students on a Schengen visa spending ~1.5 months in Oslo (e.g. exchange or internship). The plan maximizes countries visited while keeping weekdays free for university commitments.

**Key strategies:**
- Overnight FlixBus to Stockholm saves accommodation cost
- Copenhagen + Malmö combo via Øresund Bridge = 2 countries, 1 flight
- a&o and Generator hostels for best price/quality ratio
- easyJet and Norwegian Air for cheapest Oslo routes
